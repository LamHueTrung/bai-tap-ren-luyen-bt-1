Problem

Reverse A given Number / Reverse a Number . (Tiếng việt: Đảo ngược 1 số nhất định / 1 số bất kì.) => Nguồn: https://www.programiz.com.

Suggestions

+ Sử dụng vòng lặp While. + Lấy từng phải tử sau đó đảo vị trí nó lại.

Nhắc lại kiến thức

Vòng lặp While. 
                        
while (bieu_thuc) {
    khối lệnh;
}

Code mẫu: 

                        
#include < stdio.h >  
int main() {
    int n, t;
    printf("nhap vao n");
    scanf("%d", &n);
    int ans = 0;
    while(n != 0) {
        t = n%10;
        ans = ans*10 + t; 
        n /= 10;
    }
    printf("kết quả sau khi đảo: %d", ans);
    return 0;
}
